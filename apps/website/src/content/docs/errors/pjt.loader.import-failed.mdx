**What**: The dynamic `import()` of your blueprint file threw. Common causes: syntax errors, missing dependencies, or a TypeScript feature the loader can't handle.

**How to fix**: Run `node --experimental-strip-types .pjt.ts` to see the raw error, or run with the `tsx` CLI to confirm the file parses. Install any missing packages.
