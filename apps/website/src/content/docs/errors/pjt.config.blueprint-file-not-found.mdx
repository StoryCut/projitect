**What**: The CLI looked for `.pjt.ts` (or whatever `--blueprint-file` points at) and didn't find it.

**How to fix**: Run `pjt init` to create a starter `.pjt.ts`, or pass `--blueprint-file <path>` if your blueprint file lives elsewhere.
